# Agrinho 2026

## Tema

Agro forte, futuro sustentável: equilíbrio entre produção e meio ambiente.

## Assunto Escolhido

Uso consciente da água na agricultura.

## Sobre o Projeto

Este projeto foi desenvolvido para o Concurso Agrinho 2026 na categoria Programação Web.

A aplicação apresenta informações sobre o uso responsável da água na agricultura e demonstra como práticas sustentáveis podem aumentar a produtividade e preservar os recursos naturais.

## Objetivo

Conscientizar sobre a importância da economia de água no campo e mostrar como a sustentabilidade pode contribuir para um futuro melhor.

## Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript
* Inteligência Artificial como apoio ao desenvolvimento

## Funcionalidades

* Página informativa.
* Layout responsivo.
* Calculadora de economia de água.
* Conteúdo educativo sobre sustentabilidade.
